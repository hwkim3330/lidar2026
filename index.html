Record / Replay (Ouster SDK)

PCAP 기록은 Record 페이지에서 제어합니다. 파일은 data/recordings/*.pcap 에 저장되며 Ouster 센서 메타데이터 JSON이 sidecar로 함께 기록됩니다.

외부 SDK 뷰어: .venv/bin/ouster-cli source data/recordings/<file>.pcap viz

API

  • GET /api/config — lidar list
  • GET /api/lidar/stats — live packet stats
  • GET /api/lidar/profile/lidar-a — jitter/period profile
  • POST /api/record/start /stop, GET /api/recordings
  • POST /api/replay/start /stop
